# number-to-word_vietnamese

A Clojure library for number to Vietnamese word conversion.

## Usage

```clojure
(require '[n2w-vi.core :refer :all])


(number->word 9000000000000000000000000000000000)
;; => "chín triệu tỷ tỷ tỷ"

```

Enjoy!
